PROJECT OVERVIEW
----------------------------------
EcoGolf ERP is a bilingual (English / Arabic) prototype web system with modules for:
- Warehouse Management
- Accounting and Finance
- Payroll and HR
- Projects
- Reports
- User Logs
- System Settings

It includes:
- Light / Dark Mode Toggle
- English / Arabic (RTL) Language Switch
- Splash Screen and Login Page (simulated)
- Dashboard with Bilingual Cards and Charts
- Mobile / Tablet Responsive Layout

DEPLOYMENT INSTRUCTIONS
----------------------------------

1. Log in to your GitHub account.
2. Create a new public repository named: ecogolf-erp
3. On your computer, prepare a folder with the structure above.
4. Copy all files into it (HTML, CSS, JS, etc.).
5. In the repository, click “Add file → Upload files.”
6. Drag your project folder’s contents into the upload box.
7. Scroll down and click “Commit changes.”

Then enable GitHub Pages:
1. Go to “Settings → Pages.”
2. Under “Build and deployment,” set:
   Source: Deploy from a branch
   Branch: main
   Folder: /(root)
3. Click “Save.”

GitHub will automatically build your site.
After 2–5 minutes, your ERP will be live at:

   https://<your-username>.github.io/ecogolf-erp
